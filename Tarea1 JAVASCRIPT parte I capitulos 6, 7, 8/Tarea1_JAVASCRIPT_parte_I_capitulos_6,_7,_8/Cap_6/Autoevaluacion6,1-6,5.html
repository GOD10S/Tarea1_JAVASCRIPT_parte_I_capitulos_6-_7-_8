6.1 Complete las siguientes oraciones

  1. a) // comienza un comentario de una sola línea.
  2. b) Toda instrucción de JavaScript debe terminar con un ; (punto y coma).
  3. c) La instrucción if se utiliza para tomar decisiones.
  4. d) El objeto window muestra diálogos alert y prompt.
  5. e) Las palabras reservadas están reservadas para uso de JavaScript.
  6. f) Los métodos write y writeln del objeto document escriben texto en un documento HTML5.

6.2 Indique si cada enunciado es verdadero o falso

  • a) Falso. Los comentarios no provocan acciones al ejecutar el script; se usan para documentar y mejorar la legibilidad.
  • b) Falso. JavaScript distingue entre mayúsculas y minúsculas, por lo que las variables son diferentes.
  • c) Verdadero.
  • d) Falso. Los operadores *, / y % tienen el mismo nivel de precedencia; + y - tienen menor precedencia.
  • e) Falso. La función parseInt convierte una cadena en un valor entero.

6.4 Identificación y corrección de errores

    // a) Corrección de la instrucción if
    if (c < 7) {
      window.alert("c es menor que 7");
    }

    // b) Corrección del operador relacional
    if (c >= 7) {
      window.alert("c es igual o mayor que 7");
    }

6.5 Instrucciones y comentarios

    // a) Calcular el producto de tres enteros
    // Esta secuencia de comandos calcula el producto de tres enteros.

    // b) Declarar variables
    var x, y, z, resultado;

    // c) Declarar variables adicionales
    var xVal, yVal, zVal;

    // d-f) Pedir valores al usuario y almacenarlos en variables
    xVal = window.prompt("Escriba el primer entero:", "0");
    yVal = window.prompt("Escriba el segundo entero:", "0");
    zVal = window.prompt("Escriba el tercer entero:", "0");

    // g-i) Convertir las cadenas en enteros
    x = parseInt(xVal);
    y = parseInt(yVal);
    z = parseInt(zVal);

    // j) Calcular el producto
    resultado = x * y * z;

    // k) Mostrar el resultado en el documento HTML
    document.writeln("< h1 >El producto es " + resultado + "< /h1 >");
