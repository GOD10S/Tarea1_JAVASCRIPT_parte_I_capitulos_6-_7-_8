Ejercicio 6.13: Ecuación y = a * x^3 + 7

¿Cuáles de las siguientes instrucciones de JavaScript son correctas para la ecuación y = a * x^3 + 7?

  1. a) y = a * x * x * x + 7;

    Correcta: Esta instrucción representa correctamente la ecuación a * x^3 + 7.

  2. b) y = a * x * x * (x + 7);

    Incorrecta: Esta instrucción agrega 7 dentro del producto, lo que no es parte de la ecuación original.

  3. c) y = (a * x) * x * (x + 7);

    Incorrecta: El término (x + 7) altera la ecuación original.

  4. d) y = (a * x) * x * x + 7;

    Correcta: Esta instrucción es equivalente a a * x^3 + 7.

  5. e) y = a * (x * x * x) + 7;

    Correcta: Esta instrucción representa correctamente a * x^3 + 7.

  6. f) y = a * x * (x * x + 7);

    Incorrecta: El término (x * x + 7) no corresponde a la ecuación dada.
