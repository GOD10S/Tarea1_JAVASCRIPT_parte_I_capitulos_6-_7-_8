Ejercicio 6.11: ¿Qué Aparece en los Diálogos de Alerta?

  1. a) window.alert( "x = " + x );

    Resultado: x = 2

  2. b) window.alert( "El valor de x + x es " + ( x + x ) );

    Resultado: El valor de x + x es 4

  3. c) window.alert( "x=" );

    Resultado: x=

  4. d) window.alert( ( x + y ) + " = " + ( y + x ) );

    Resultado: 5 = 5
